# Unity 3D单机多人练习Demo
## 项目介绍
独立开发5周，支持PC/安卓双端运行，包含单机闯关、局域网多人联机完整玩法，集成手游工业化热更新、AI战斗、移动端适配、全套能力。
## 核心技术栈
- 引擎：Unity C#、UGUI、动画状态机、对象池性能优化
- 热更新：AB包资源热更（UI/贴图/音效）、ILRuntime代码热更（开火逻辑、战斗参数动态更新）
- 网络：NetcodeForGameobject，状态同步，PC/手机跨设备联机，本地预测优化延迟
- AI：敌人巡逻、视野索敌、追击、射击、死亡完整状态机AI
- 移动端：多分辨率UI自适应、虚拟摇杆触屏、安卓APK打包适配
- 拓展：UE5基础、C++、图形学基础（个人拓展学习）
## 项目功能演示
1. 单机模式：角色移动/冲刺/下蹲/下趴/跳跃/射击，完整敌人对战体系
2. 热更新演示：无需重打包，动态替换UI资源、修改战斗逻辑以及参数
3. 跨端联机：局域网PC与安卓手机同步角色位置、伤害血量
4. 性能优化：对象池复用所有特效，减少GC卡顿
## 运行说明
1. 克隆仓库，用对应Unity版本打开工程
2. AB包资源热更需自行执行打包工具生成资源包
3. 联机模式需保证两台设备处于同一虚拟局域网
